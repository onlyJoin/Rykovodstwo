Переместите файл vzlom.exe на компьютер, и спрячьте среди папок

            В этой же папке с exe, создайте папку data
        и поместите в него файл со звуком,
        переменуйте этот файл в "agogo-bells__025_mezzo-forte_struck-singly.mp3"

        Теперь при запуске приложения, будет звучать постоянно звук,
        при этом возможности закрыть приложение нет


        Для того чтобы закрыть это приложение (выключить звук),
        переходим в диспечер задач и находим vzlom.exe, снимаем задачу,
        теперь приложение закрылось!
    

    Для того чтобы сделать запуск приложения при автозапуске,
    достаточно запустить add_auto.exe и записать адрес нахождения vzlom.exe,
    теперь наше приложение автоматически запускается при запуске window
    

    Для того чтобы отключить автозапуск,
    Перейдем в диспечер задач => Автозагрузка приложений,
    найдем строку open и отключим ее,
    все автозапуск отключен
